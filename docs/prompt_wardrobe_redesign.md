# Гардероб: визуальная навигация

## Проблема
Текущий `_show_wardrobe_page()` — текстовый список: "🟤 футболка серый 👍".
Нечитаемо при 6 вещах, невозможно при 50+. Нет фото, нет фильтров.

## Новый подход: обзор + grid + карточка вещи

### Экран 1: Обзор (текст + inline кнопки)
Одно текстовое сообщение. edit_message_text для всех переключений.

```
👧 Алиса · 25 вещей | 👩 Катя · 18 вещей     ← tabs, если "для обоих"

👗 Гардероб Алисы

🧥 Верхняя — 3
👚 Верх — 6
👖 Низ — 4
👟 Обувь — 4
👗 Платья — 2
🧦 Базовый — 6

[🧥 Верхняя] [👚 Верх] [👖 Низ]
[👟 Обувь] [👗 Платья] [🧦 Базовый]
[❄ Зима] [🌸 Весна] [☀ Лето] [🍂 Осень]
[👩 Показать Катю]                          ← только для "обоих"
```

**Сезонный фильтр:** нажатие "❄ Зима" = edit_message_text.
Числа пересчитываются (только зимние). Кнопка подсвечена.
Категории с 0 вещей — не показывать. "Все сезоны" для сброса.

**Owner:** tabs видны сразу. Переключение = edit того же сообщения.
Для single owner — без tabs.

### Экран 2: Grid категории (фото-картинка)
Нажатие категории → новое сообщение с картинкой + inline кнопки.

**Grid-картинка** (Satori или PIL fallback):
- 3x3 ячейки, max 9 на страницу
- Каждая ячейка: thumbnail фото или пастельная заливка по цвету
- Подпись внутри ячейки (overlay): цвет ("розовая", "белая пр.")
- Заголовок: "👚 Верх · Алиса · 6 вещей"

**Кнопки:**
```
[1 розовая] [2 белая пр.] [3 серая пр.]
[4 зелёная] [5 бежевая] [6 белая]
[← Гардероб] [1/1]
```
Подписи ≤14 символов. Пагинация: deleteMessage + sendPhoto.
deleteMessage в try/except (48h limit), fallback = новое.

**Callback_data:**
```
w:overview                         ← обзор
w:owner:{owner_id}                 ← owner
w:season:{season}                  ← фильтр
w:cat:{category}:p{page}           ← категория
w:item:{index}                     ← вещь (index 1-9 на странице)
w:del:{item_id}                    ← удалить
w:del_confirm:{item_id}            ← confirm
w:season_edit:{item_id}:{season}   ← сезон вещи
```
UUID не влезает в 64 bytes → numeric index 1-9 на странице.
Mapping index→UUID в context.user_data["wardrobe_items"].

### Экран 3: Карточка вещи
Фото через file_id + caption + inline кнопки.
```
Футболка серая с принтом
Верх · Лето, Весна · серый
Добавлена 15 мар

[🗑 Удалить] [Сезон] [← Верх]
```
"Удалить" → confirm: [Да, удалить] [Отмена]
"Сезон" → [Зима] [Весна] [Лето] [Осень] [Все]

## Thumbnails
1. getFile(file_id) → download
2. Resize 150x150
3. Кеш: Redis `thumb:{file_id}` base64, TTL 24h. Или /tmp/thumbs/
4. В grid: Satori img base64 или PIL paste
5. Нет фото → пастельная заливка по цвету
