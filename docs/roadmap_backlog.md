## v1.1b (апрель)

### Стилист
- [ ] Комментарий Касси = позитивный framing: "розовый тебя освежает", НЕ "тебе не идёт оранжевый". Никаких запретов, только рекомендации.
- [ ] Фото для стайл-квиза: каждый образ ЧИСТО одного стиля. "Немного Classic + Casual" = юзер не видит разницу.

### UX лид
- [ ] Fallback ручной выбор цветотипа (4 кнопки с описанием) обязателен если Vision confidence < 0.6
- [ ] Карточка результата цветотипа = shareable момент, предусмотреть кнопку шеринга

### Поведенческий психолог
- [ ] Прогресс внутри стайл-квиза: "3/10", "7/10 — почти!" — чтобы не бросила на середине
- [ ] Результат квиза = identity moment. Юзер идентифицирует себя через стиль → commitment

### Блогер
- [ ] Фото для квиза: НЕ stock. AI-generated в едином стиле или flat lay. Все на одном фоне.
- [ ] Карточка результата "Elegant Classic" = #1 viral asset. Дизайн как в журнале.

### Solo CTO
- [ ] Квиз: НЕ ConversationHandler (10 states громоздко). Один handler + quiz_step в context.user_data
- [ ] Контент квиза = bottleneck (20 фото, 1-2ч генерации). Делать параллельно с кодом.

### Лена (ЦА)
- [ ] Free чат: 3 сообщения/день, не 1. Минимальный диалог, иначе фрустрация.

### Monetization
- [ ] Trial end: soft ask "Спасибо 💕 Хочешь продолжить?" не "доступ закрыт"
- [ ] Первым юзерам: "$5/мес навсегда" в первые 24ч после trial = валидация + lock-in
